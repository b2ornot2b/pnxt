Beyond Phase 4's prototype, pnxt's roadmap extends toward a complete runtime for agent-native programming. These goals represent the full realization of the [theoretical foundations](/pnxt/concepts/foundations/).

---

## Medium-Term Goals

### Bridge Grammar Implementation

Build the constrained-decoding JSON schema that forces LLMs to output valid VPIR nodes:
- Define the complete Bridge Grammar schema
- Implement constrained decoding integration
- Validate that outputs are structurally correct VPIR

### Natural Language Protocol Design

Formalize communication patterns for agent-to-agent collaboration:
- Negotiation protocols
- Task delegation patterns
- Conflict resolution through natural language
- Shared vocabulary and conventions

### VPIR Compiler/Interpreter

Execute Verifiable Programmatic Intermediate Representation:
- Parse VPIR nodes into executable form
- Verify proof certificates at each step
- Generate execution traces
- Integration with SMT solver verification

### Tree-sitter DKB Integration

Knowledge graph-based codebase representation:
- Parse codebases into graph structure
- Semantic queries over the graph
- Dependency analysis and impact assessment
- Integration with agent memory for codebase understanding

## Long-Term Goals

### LLMbda Calculus Runtime

Full lambda calculus with Information Flow Control:
- Dynamic security labels on all values
- Noninterference enforcement
- Formal protection against prompt injection
- Mathematical proof of security properties

### SMT Solver Integration

Z3/CVC5 for constraint satisfaction and formal verification:
- Autoformalization of agent claims
- Constraint checking on generated code
- Invariant verification
- Automated theorem proving for agent reasoning

### Full Dataflow Process Network Engine

Actor-based execution with FIFO channel communication:
- Define and execute DPN graphs
- Non-blocking, unidirectional channels
- Deterministic parallel execution
- Visual node-graph decompiler for human oversight

### Multi-Agent Orchestration at Scale

Enterprise deployment with full governance:
- Federated trust across organizations
- Multi-project, multi-team coordination
- Compliance and audit at scale
- Self-organizing agent teams

### Community & Ecosystem

Open standards and adoption tooling:
- Open specification for ANP protocols
- Reference implementations
- Developer tooling and IDE integrations
- Community governance and contribution model
